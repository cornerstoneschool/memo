# 📸 Memo Pics 2025

A beautiful, glassmorphic photo gallery for farewell events. Built with vanilla HTML, CSS, and JavaScript - deployable on GitHub Pages.

## ✨ Features

- 🎨 Gold & Black Glassmorphic Theme
- 🔐 Super Admin & Teacher Authentication
- 📷 Dynamic Photo Gallery with Masonry Layout
- ❤️ Reaction System (Love, Laugh, Emotional, Fire, Clap, Star)
- 📥 Photo Download (Individual & Bulk)
- 🗂️ Custom Categories
- 🔒 Teachers-Only Content
- 💬 WhatsApp Sticker Maker
- 🎬 Slideshow Generator
- 📊 Analytics Dashboard
- 📱 Fully Responsive
- 💾 Local Storage Based (No Backend Required)

## 🚀 Deployment

### GitHub Pages

1. Fork this repository
2. Go to Settings > Pages
3. Select "main" branch and "/ (root)" folder
4. Save and wait for deployment
5. Access at `https://yourusername.github.io/memo-pics-2025`

### Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/memo-pics-2025.git

# Open in browser
cd memo-pics-2025
# Open index.html in your browser
